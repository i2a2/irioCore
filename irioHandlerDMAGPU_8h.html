int irio_setUpDMAsTtoGPU ( irioDrv_t *  p_DrvPvt,
int  depth,
TStatus *  status 
)

Set up DMAs.

Configure GPU memory and FPGA registers for DMA transfer. First, allocate GPU memory for DMA transfer and clean up of possible trash data. Then set. The DMA channel should be activated by the user once the initialization has finished

This method calls calcADCValue() to configure AD/DA conversion values among other fields.

See also
{calcADCValue}
Parameters
[in,out]p_DrvPvtPointer to the driver session structure
[in]depthNumber of elements in buffer. Because the total buffer size in bytes must be a multiple of 64 KiB, depth in elements must be a multiple of: 8192 for 64-bit elements (U64, I64), 16384 for 32-bit elements (U32, I32), 32768 for 16-bit elements (U16, I16), or 65536 for 8-bit elements (U8, I8, Bool).
[out]statusWarning and error messages produced during the execution of this call will be added here.
Returns
TIRIOStatusCode result of the execution of this call.
